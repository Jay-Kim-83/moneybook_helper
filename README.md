# 가계부 작성 도우미 (Moneybook)

은행·카드·고정 지출을 관리하고 월별 결제 내역을 전월과 비교하는 가계부 웹 애플리케이션입니다.
별도 의존성 없이 Node.js 내장 모듈만으로 동작하며, 데이터는 로컬 JSON 파일에 저장됩니다.

## 실행 방법

```bash
node server.js
```

브라우저에서 `http://localhost:3000` 접속

## 주요 기능

- **은행 관리** — 은행명·계좌번호·비밀번호 힌트 등록/수정/삭제
- **카드 관리** — 카드사·카드번호·CVC·연결 은행 등록/수정/삭제
- **고정 지출** — 카드 외 정기 지출 등록/수정/삭제
- **은행별 카드 보기** — 은행 섹션별 연결된 카드 표시
- **이번달 결제**
  - 은행 잔액 입력
  - 카드 결제 금액 입력 및 전월 대비 증감 비교
  - 고정 지출 자동 등록 (금액 수정/항목 추가 가능)
  - 총 잔액·결제·지출·예상 잔액 실시간 요약

## 기술 스택

- 백엔드: Node.js (내장 `http` 모듈, 의존성 없음)
- 프론트엔드: Tailwind CSS, SweetAlert2 (CDN)
- 저장소: 로컬 JSON 파일 (`data/db.json`)

## 데이터 보안

`data/` 폴더에는 계좌번호·비밀번호 힌트 등 민감 정보가 저장되므로 `.gitignore`로 제외됩니다.
저장소에는 코드만 포함되며 개인 금융 데이터는 업로드되지 않습니다.
